# PDF_Scanner

PDF_Scanner is a PDF action suite using Python, which takes a PDF file and modifies it with the following actions:
- add_blank
- decrypt
- encrypt
- get_range
- merge
- read_write
- remove_blank
- remove_page
- rotate
- scan_form
